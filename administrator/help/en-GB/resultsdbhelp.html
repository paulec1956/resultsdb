Managing Results: Add, modify and delete

Click on Results in the list on the left hand side of the page:
  • Add a result
  • Add a number of results for a race
  • Delete a result
  • Modify a result
  • Check how the results look

Add a result

Make sure you have already entered the race for the result(s) you want to enter

  1. Click on the green New button to open the input screen for a result.
  2. This opens the form to add a result:
  3. Enter the result details:
    4. Field Details
    Race: Select a race from the drop down list. You can type into the selection box to filter the races shown
    Date: Choose the date using the date picker tool
    Runner: Choose the runner name from the drop down list
    Position: Enter the position in the race overall
    Time: Enter the result time in format hh:mm::ss or mm:ss as apropriate
    Agegrade: Enter the % age grade if available, otherwise leave blank.
    Pb: Leave blank if the result isn't a PB, other wise enter PB
    Agecat: Enter age category from (F, F30, F35… M, M30, M35,… etc)
    Catposition: Enter position in category e.g. 3/21
    Comment: Add any relevant comments

  4. Click Save and Close to save the result.

Add a number of results for a race

Whe you have to load a number of results for the same race it can be quicker if you follow this method

  1. Add the first result: Click on the green New button to open the input screen for a result.
  2. This opens the form to add a result:
  3. Enter results details
  4. Click Save to save the result, leaving the window open.
  5. Click Save as Copy to create another result for the same race

    6. The screen will stay with the result details

  6. For the next result, leave the Race and date unchanged, but edit the other fields for the result

    7. Change the Runner name, position etc

  7. Click Save to save the result, leaving the window open.
  8. If you have another result to enter, follow steps 5, 6 and 7 again
  9. If you have no more results to enter for this race, click Save and Close

Delete a Result

  1. In the list of races select the race to be deleted by ticking the box on the left hand side
  2. Click the Trash button

Modify a result

  1. Select the result to be modified and click the Edit button.
  2. This will bring the result details form to be changed.
  3. Click Save & Close to save changes or Close to abort the changes.
